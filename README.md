# OrthodoxScripture

The largest collection of Easter Orthodox scripture in one website

## Features

- All of the EO Biblical Canon with original language side by side with literal translation
- Church Fathers commentaries on individual verses
- All of the 7 major Ecumenical Councils plus the other local Councils
- A web version of "Anti-Nicean Church Fathers" and "Nicean and Post-Nicean Church Fathers" for deeper insight on the Early Church
